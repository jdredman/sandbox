Usage Guidelines

Hierarchy

Use Display for hero messages
Use Headlines for major sections
Use Body for readable content
USE LABELS FOR ACTIONS

Establish clear information hierarchy with consistent type scale usage.

Readability

Keep line lengths between 45-75 characters for optimal readability. This ensures comfortable reading without losing your place.

Maintain appropriate line lengths and spacing for comfortable reading.

Emphasis

Use semibold weight for emphasis within body text.
Use color for highlighting important information.

Create emphasis through weight and color, not size changes.

Accessibility

Ensure sufficient contrast between text and background colors for all users.

Follow WCAG guidelines for color contrast and font size minimums.

Implementation

CSS Classes

Use Material 3 typescale classes for consistent typography across your application.

<!-- Display Typography --> <h1 class="md-typescale-display-large">Hero Headline</h1> <h2 class="md-typescale-display-medium">Section Header</h2> <h3 class="md-typescale-display-small">Page Title</h3> <!-- Headline Typography --> <h4 class="md-typescale-headline-large">Major Heading</h4> <h5 class="md-typescale-headline-medium">Section Heading</h5> <h6 class="md-typescale-headline-small">Card Header</h6> <!-- Body Typography --> <p class="md-typescale-body-large">Primary content text</p> <p class="md-typescale-body-medium">Secondary content text</p> <p class="md-typescale-body-small">Caption or helper text</p> <!-- Label Typography --> <span class="md-typescale-label-large">Button Text</span> <span class="md-typescale-label-medium">Tab Label</span> <span class="md-typescale-label-small">Small Label</span>

Custom Font Integration

/* Load Gibson via TypeKit */ @import url('https://use.typekit.net/euz0vgw.css'); /* Define font tokens */ :root { --font-family-gibson: "gibson", "Helvetica Neue", Helvetica, Arial, sans-serif; --font-family-base: var(--font-family-gibson); --font-weight-normal: 400; --font-weight-semibold: 600; --font-weight-bold: 700; } /* Override Material 3 font tokens */ :root { --md-ref-typeface-brand: var(--font-family-gibson); --md-ref-typeface-plain: var(--font-family-gibson); --md-sys-typescale-display-large-font: var(--font-family-gibson); /* ... continue for all typescale tokens ... */ }
