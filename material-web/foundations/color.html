Material 3 Semantic Colors

Our Ramsey colors are mapped to Material 3 semantic roles for consistent component theming and accessibility.

Primary

Main brand actions

Uses Blue 50 for high-emphasis actions
--md-sys-color-primary --md-sys-color-on-primary

Secondary

Supporting actions

Uses Teal 50 for medium-emphasis actions
--md-sys-color-secondary --md-sys-color-on-secondary

Tertiary

Accent elements

Uses Purple 50 for accent and contrast
--md-sys-color-tertiary --md-sys-color-on-tertiary

Error

Error states

Uses Red 60 for errors and warnings
--md-sys-color-error --md-sys-color-on-error

Surface

Background surfaces

Uses White and Gray 10 for content areas
--md-sys-color-surface --md-sys-color-on-surface

Accessibility Guidelines

Contrast Standards

  • All color combinations meet WCAG AA standards
  • Text on backgrounds maintains 4.5:1 contrast ratio
  • UI elements maintain 3:1 contrast ratio
  • Large text maintains 3:1 contrast ratio

Color Independence

  • Never rely on color alone to convey information
  • Use icons, text labels, or patterns alongside color
  • Provide alternative indicators for status and state
  • Test with color vision deficiency simulators

Best Practices

  • Use semantic color tokens rather than base colors
  • Test in both light and dark modes
  • Maintain color meaning consistency across the experience
  • Consider cultural color associations for global audiences
