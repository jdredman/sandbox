Best Practices

Do

  • Use for binary on/off settings
  • Provide clear, descriptive labels
  • Include helpful descriptions
  • Show immediate effect when toggled
  • Group related switches logically

Don't

  • Use for actions that require confirmation
  • Mix switches with other selection controls
  • Use vague or confusing labels
  • Make destructive changes without warning
  • Disable without explanation

Accessibility

  • Include proper ARIA labels
  • Ensure 44px minimum touch target
  • Support keyboard navigation
  • Announce state changes
  • Provide sufficient color contrast
