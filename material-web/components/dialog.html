Best Practices

Do

  • Use clear, descriptive titles
  • Keep content concise and scannable
  • Place primary action on the right
  • Use dismissible dialogs for non-critical tasks
  • Include proper focus management

Don't

  • Overuse dialogs for simple messages
  • Stack multiple dialogs
  • Use vague or unclear button labels
  • Include too much information
  • Block critical workflows unnecessarily

Accessibility

Focus Management

  • Focus moves to dialog when opened
  • Focus returns to trigger when closed
  • Tab cycles within dialog content
  • First focusable element receives focus

Keyboard Navigation

  • Escape - Close dismissible dialogs
  • Tab - Move to next element
  • Shift + Tab - Move to previous element
  • Enter - Activate focused button

Screen Reader Support

  • Dialog role is properly announced
  • Title and description are labeled
  • Modal state is communicated
  • Content changes are announced
