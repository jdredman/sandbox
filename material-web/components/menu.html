Best Practices

Do

  • Group related actions logically
  • Use dividers to separate sections
  • Include keyboard shortcuts where appropriate
  • Provide clear, descriptive labels
  • Use consistent icons throughout

Don't

  • Create overly long menu lists
  • Use ambiguous or unclear labels
  • Mix different interaction patterns
  • Forget to handle disabled states
  • Ignore mobile touch targets

Accessibility

  • Support keyboard navigation
  • Provide proper ARIA labels
  • Ensure sufficient contrast
  • Support screen readers
  • Include focus indicators
