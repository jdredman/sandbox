Best Practices

Do

  • Use consistent item heights within a list
  • Include relevant leading icons or avatars
  • Provide clear visual hierarchy
  • Use sectioning for logical grouping
  • Support keyboard navigation

Don't

  • Mix different list item types inappropriately
  • Use lists for single items
  • Overcrowd with too much information
  • Forget hover and focus states
  • Use poor contrast for secondary text

Accessibility

  • Use semantic list markup (ul, ol, li)
  • Provide proper ARIA labels
  • Ensure keyboard navigation works
  • Support screen readers properly
  • Maintain focus indicators
