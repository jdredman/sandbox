Usage Guidelines

✅ Do

Use filled buttons for primary actions like "Submit" or "Continue"

❌ Don't

Don't use multiple filled buttons in the same context

✅ Do

Use consistent button hierarchies across your interface

❌ Don't

Don't use buttons for navigation between pages
