Accessibility

Labels and Descriptions

  • Every checkbox has a clear, descriptive label
  • Labels are clickable to toggle the checkbox
  • Group labels describe the checkbox collection
  • Instructions are provided when needed

Keyboard Navigation

  • Tab - Move to next checkbox
  • Shift + Tab - Move to previous checkbox
  • Space - Toggle checkbox state
  • Arrow keys work in checkbox groups

Screen Reader Support

  • State is announced (checked/unchecked/mixed)
  • Labels are read with checkbox
  • Group context is provided
  • Changes are announced immediately
