Best Practices

Do

  • Keep content focused on a single subject
  • Use consistent card dimensions in grids
  • Include clear hierarchy with titles and body text
  • Provide meaningful actions
  • Use appropriate elevation for context

Don't

  • Overload cards with too much content
  • Use cards for single lines of text
  • Mix different card types inappropriately
  • Forget to provide touch/click feedback
  • Use low contrast for text

Accessibility

  • Ensure sufficient color contrast
  • Provide alt text for images
  • Use semantic HTML structure
  • Support keyboard navigation
  • Include proper ARIA labels
