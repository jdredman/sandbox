Accessibility

Labels and Descriptions

  • Always include aria-label for screen readers
  • Use descriptive labels that explain the action
  • Indicate toggle states in labels
  • Group related buttons with roles

Keyboard Navigation

  • Tab - Move to next button
  • Shift + Tab - Move to previous button
  • Enter or Space - Activate button
  • Arrow keys work in button groups

Visual Indicators

  • 44px minimum touch target size
  • Clear focus indicators
  • High contrast color combinations
  • State changes are visually distinct
