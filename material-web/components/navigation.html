Best Practices

Do

  • Use consistent navigation patterns throughout your app
  • Provide clear visual indicators for current location
  • Make navigation accessible with keyboard support
  • Choose appropriate navigation for screen size
  • Group related navigation items logically

Don't

  • Mix different navigation patterns unnecessarily
  • Create overly complex navigation hierarchies
  • Use unclear or ambiguous labels
  • Forget to indicate the current page/section
  • Ignore mobile touch target sizing

When to Use

  • Bottom Navigation: Mobile apps with 3-5 top-level destinations
  • Navigation Rail: Tablet and desktop apps with vertical space
  • Navigation Drawer: Apps with many destinations or complex hierarchies
  • Top App Bar: All apps for consistent branding and actions
  • Breadcrumbs: Deep hierarchical navigation structures
