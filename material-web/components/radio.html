Best Practices

Do

  • Use for mutually exclusive choices
  • Provide clear, descriptive labels
  • Group related options together
  • Include helpful descriptions
  • Set sensible defaults when appropriate

Don't

  • Use for multiple selections (use checkboxes)
  • Have too many options in one group
  • Use for yes/no choices (use switch)
  • Forget to include a name attribute
  • Make labels too long or complex

Accessibility

  • Use proper label associations
  • Ensure 44px minimum touch target
  • Support keyboard navigation
  • Group related options with fieldset
  • Provide clear error messages
