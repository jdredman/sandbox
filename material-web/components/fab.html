Best Practices

Do

  • Use for the primary action on a screen
  • Place in the bottom-right corner
  • Keep above other elements
  • Use meaningful icons and labels
  • Provide proper accessibility labels

Don't

  • Use for every action on a screen
  • Place multiple FABs on one screen
  • Use for destructive actions
  • Block important content
  • Change position during scroll

Accessibility

  • Always include aria-label
  • Ensure 44px minimum touch target
  • Support keyboard navigation
  • Use high contrast colors
  • Announce state changes
